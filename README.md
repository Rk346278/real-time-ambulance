A real-time emergency response system that tracks ambulance locations, predicts patient severity using AI-inspired NLP logic, and simulates smart traffic signal prioritization to reduce response time.
Features
Live ambulance GPS tracking using Socket.IO
AI-based severity scoring from nurse-entered medical notes
Optimized route suggestions using OSRM
Hospital dashboard for real-time monitoring and prioritization
Tech Stack
Frontend: React.js
Backend: Node.js, Express
Database: MongoDB
Real-Time: Socket.IO
Routing: OSRM (OpenStreetMap)
